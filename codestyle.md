# InfectStatistic-学号

- 缩进

1）作用域里的语句从第二行开始缩进一个tab键

2）左括号跟在行末

- 变量命名

1）变量名称全小写

2）尽量使用英文单词

- 每行最多字符数

1）每行最多字符数为80个

- 函数最大行数

1）每个函数最大150行

- 函数、类命名

1）函数命名采取单词首字母大写的形式

2）类的命名与函数的命名相同

- 常量

1）常量全部大写

2）不允许出现未定义过的常量

- 空行规则

1）函数与函数之间空一行

- 注释规则

1）单行注释为//

2）多行注释用/*   */

- 操作符前后空格

1）操作符之前均有空格例如：`int a = 19;`
